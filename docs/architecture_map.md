# Architecture Map

```text
Program
  -> Virtual Address
  -> TLB / Page Table
  -> Physical Address
  -> L1 / L2 / L3 Cache
  -> DRAM
  -> Storage / File System
  -> Process / Thread / Scheduler
  -> IPC / Synchronization
  -> Kernel / Driver / Mini OS
  -> Virtualization / Isolation
```

This repository is organized so that each lab connects one piece of that path to measurable behavior.
